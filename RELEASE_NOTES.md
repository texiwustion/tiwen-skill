# Release Notes

## 0.1.0

首个公开版本。

### Added

- 技术 / 产品 / 工具 / 科研 / 项目决策的 v1 作用域
- 私有 friction scan，用来捕捉用户问题和常见默认回答之间的摩擦点
- 2～6 问的苏格拉底追问循环
- 默认硬门槛：前提不够，不给成熟问题
- `--wounded` 带伤放行模式
- `--agent` 目标改写模式
- `--cold` 冷刀模式
- 高风险题材安全卡口
- 对抗性自测样例：`tiwen-skill/references/self-test-cases.md`

### Behavior notes

- 默认产物是给真人的一句半口语成熟问题
- 不回答原问题，不做教程，不做工具或模型推荐
- 遇到混词场景时，优先钉住对象归属，不默认同义

### Known limits

- 目前最稳的是技术/项目类问题
- 对极短输入的语气仍可能偏硬，后续还可以继续调校
